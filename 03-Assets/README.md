This index provides a navigable view of reusable DMAF consulting assets.

## 8. Asset Governance Notes

All reusable DMAF assets should follow these governance rules:

1. each asset must map to an approved DMAF capability;
2. each asset must identify its practice domain;
3. each asset must identify its lifecycle stage;
4. each asset must include ownership and status;
5. each asset must be version controlled;
6. each asset should include usage guidance;
7. each asset should identify review criteria;
8. each asset should identify future Workbench potential where relevant.
