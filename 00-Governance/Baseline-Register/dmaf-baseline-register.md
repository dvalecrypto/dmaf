# DMAF Baseline Register

## 1. Purpose

The DMAF Baseline Register provides the formal record of approved framework decisions, foundational artifacts, governance checkpoints, and release-level baselines for the Databricks Modernization Advisory Framework.

The purpose of this register is to ensure that DMAF evolves as a governed body of intellectual property rather than as a loose collection of documents.

The Baseline Register supports:

* traceability;
* release governance;
* decision history;
* framework consistency;
* asset governance;
* change control;
* future Workbench indexing.
