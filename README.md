# DMAF Repository Bootstrap Package

## Files Included

- `bootstrap_dmaf.py`

## How to Use

1. Copy `bootstrap_dmaf.py` into the root of your empty DMAF GitHub repository.
2. Open a terminal in that repository.
3. Run:

```bash
python bootstrap_dmaf.py
```

4. Review the generated folders and files.
5. Commit the result:

```bash
git add .
git commit -m "Initialize DMAF repository structure"
git push
```

## What It Creates

The script creates the DMAF folder hierarchy, top-level repository files, folder README files, metadata templates, GitHub issue and PR templates, CODEOWNERS placeholder, and Workbench CLAUDE.md placeholder.
